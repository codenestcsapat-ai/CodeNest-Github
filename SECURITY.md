# Security Policy

## Supported Versions

This repository contains the public CodeNest website.

Security updates are applied to the currently published version of the site and the active development preview.

| Version | Supported |
| ------- | --------- |
| Current live site | Yes |
| Active V2 preview | Yes |
| Older drafts / archived branches | No |

## Reporting a Vulnerability

If you discover a security vulnerability, please do not open a public issue.

Instead, report it privately through GitHub's private vulnerability reporting feature, or contact us at:

info.codenest.hu@gmail.com

Please include:

- a short description of the issue
- the affected URL or file
- steps to reproduce the issue
- screenshots or logs if useful

We will review the report and respond as soon as reasonably possible.

If the issue is accepted, we will work on a fix before publicly discussing details.
If the issue is declined, we will explain why where possible.
